# Beltis Test

A Flutter application for the interview test of Beltis.

## Installation

1. Clone the repository.
2. Run `flutter pub get` to install the dependencies.
3. Run the app using `flutter run`.

## Features

- Login Page
- List of Packages and Box Info
- Package details

## Tests

Run the tests using:
```sh
flutter test
```
